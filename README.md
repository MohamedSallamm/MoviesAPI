## Overview
Movies Library API is a RESTful API built with ASP.NET Core that allows users to manage a movie collection. 
The API supports full CRUD operations and is designed to be consumed by web or mobile applications.

Key functionalities:
- Add, update, delete, and retrieve movies.
- Categorize movies by genre and release year.
- JSON-based API responses for easy integration.
- Built with scalability and clean architecture in mind.

## Technologies Used
- ASP.NET Core 8 Web API
- C#
- Entity Framework Core
- SQL Server
- Swagger / OpenAPI for API documentation


## Features
- **CRUD operations** for movies.
- **Filtering and sorting** by genre, release year, or rating.
- **Validation** for required fields and data consistency.
- **API documentation** with Swagger for easy testing.
